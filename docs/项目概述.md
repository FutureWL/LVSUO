# LVSUO 律所项目门户 · 项目概述

## 一、项目背景

随着 FutureWL 律师事务所业务规模扩大,案件数量、客户资源、内部文档持续增长,传统的 Excel + 邮件 + 共享文件夹模式已无法满足协同办公需求,亟需一套统一的内部项目门户系统。

## 二、项目目标

1. **统一入口**:所有案件、项目、客户信息集中管理
2. **高效协同**:律师、助理、行政人员在线协作
3. **知识沉淀**:案例、模板、法律法规知识库可复用
4. **数据驱动**:工时、营收、案件数据可视化分析

## 三、用户角色

| 角色 | 主要职责 |
|------|---------|
| 合伙人 | 案件分配、营收审批 |
| 主办律师 | 案件推进、客户沟通 |
| 律师助理 | 文档整理、任务执行 |
| 行政人员 | 立案登记、计费收费 |
| 客户(可选) | 案件进度查询、文件签收 |

## 四、迭代计划

- **V1.0**:案件管理 + 客户管理 + 基础文档
- **V1.1**:工时记录 + 计费账单
- **V1.2**:数据看板 + 知识库
- **V2.0**:客户门户 + 移动端 App
