> **交付原则**
>
> - **文档先行**:架构 / 接口 / 数据 / 状态机 文档先于代码,作为多人协作的契约
> - **代码即文档**:关键模块 README + 注释 + TypeDoc / JSDoc
> - **可独立运行**:每个交付物必须可独立验证,不允许"半成品"
> - **可回滚**:部署包支持一键回滚到上一版本
> - **可观测**:所有服务带 metrics / trace / log 三件套
