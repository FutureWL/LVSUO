# 第 11 章 · 前端页面清单

> 本章定义 LM Unity · Counsel + 律时 的 **4 类前端** 页面清单:律所管理端 / 律师端 / 独立律师端 / 客户端。
>
> **前端技术栈建议**:
> - 律所管理端 / 律师端:Vue 3 + TypeScript + Vite + Element Plus
> - 客户端(轻量):uni-app / Taro(支持小程序 + H5 + App)
> - 独立律师端:Vue 3 + Vite(简化版)
