# 第 14 章 · MVP 开发范围

> 本章定义 **第一阶段(MVP)** 的开发范围,目标是跑通"**合规获客 → 律时接待 → 分诊 → 产品推荐 → 报价 → 签约 → 案件作战室 → 工时 → 客户周报 → 结案复盘 → 知识沉淀**"的完整业务闭环。

---

## 14.1 第一阶段目标

先跑通上述 11 个业务环节的最小可行产品,验证商业模式与技术架构的可行性。
