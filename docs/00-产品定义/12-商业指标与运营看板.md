# 第 12 章 · 商业指标与运营看板

> 本章定义系统的 **4 大指标体系**:合规增长 / 服务交付 / 律师效率 / 律所组织能力。
> 指标对应 1.1 节"法律服务组织能力系统"的可量化评估。

---

## 12.1 合规增长指标

| 指标 | 含义 | 数据源 |
|------|------|--------|
| 线索数量 | 一定时间内新增线索总数 | `leads` |
| 合规线索率 | 通过隐私授权 / 律时初步接待的线索占比 | `leads.intake_status` |
| 分诊完成率 | 已分诊线索 / 总线索 | `intake_triages` |
| 律师确认率 | 已分诊线索中律师确认的占比 | `intake_triages.lawyer_confirmed` |
| 产品推荐率 | 已推荐服务产品的线索占比 | `leads.recommended_product_id` |
| 报价转化率 | 报价数 / 线索数 | `fee_quotes` |
| 风险揭示确认率 | 客户确认风险揭示 / 报价总数 | `fee_quotes.client_confirmed` |
| 签约转化率 | 签约数 / 报价数 | `engagements` |
| 无效线索率 | `UNSUITABLE` / `CLOSED_LOST` 占比 | `leads.intake_status` |
| 投诉率 | 投诉数 / 委托数 | `service_quality_checks` |
| 违规内容拦截率 | 被拦截 / 提交内容 | `marketing_contents.review_status` |

## 12.2 服务交付指标

| 指标 | 含义 |
|------|------|
| 首次响应时间 | 客户首次咨询到首次响应的时长 |
| 律师介入时间 | 律时接待到律师确认的时长 |
| 材料收集完成时间 | 委托签约到材料收集完毕的时长 |
| 交付准时率 | 准时交付 / 总交付 |
| 质检通过率 | 质检通过 / 抽检数 |
| 客户满意度 | 客户评分均值 |
| 客户健康分 | 13.4 健康评分均值 |
| 客户复购率 | 复购客户 / 总客户 |
| 转介绍率 | 转介绍客户 / 总客户 |

## 12.3 律师效率指标

| 指标 | 含义 |
|------|------|
| 有效工时 | 已审核工时总和 |
| 可收费工时 | 计入账单的工时 |
| 案件平均耗时 | 案件从立项到结案的平均天数 |
| 文书返工率 | 文书修改次数 / 文书数 |
| 任务逾期率 | 逾期任务 / 总任务 |
| 客户响应时长 | 律师响应客户平均时长 |
| 知识贡献数 | 律师个人入库知识卡数 |
| 案件复盘完成率 | 完成复盘的案件 / 总结案数 |

## 12.4 律所组织能力指标

| 指标 | 含义 |
|------|------|
| 统一分诊覆盖率 | 全所线索经统一分诊的比例 |
| 服务产品使用率 | 通过服务产品库的订单占比 |
| 报价卡使用率 | 委托中报价卡的使用比例 |
| 风险揭示覆盖率 | 委托中风险揭示确认的比例 |
| 客户周报覆盖率 | 客户收到周报的比例 |
| 质检抽检率 | 抽检案件 / 总案件 |
| 知识卡片入库率 | 总结案有知识卡的比例 |
| 合规内容审核率 | 推广内容经三审的比例 |

---

> **可视化建议**
>
> - **总览驾驶舱**(11.1):核心 8 指标(合规线索率 / 报价转化率 / 风险揭示确认率 / 客户健康分 / 客户满意度 / 复购率 / 违规内容拦截率 / 团队负荷)
> - **客户成功看板**:客户漏斗 + 健康分分布 + 投诉热力
> - **案件风险看板**:案件状态机分布 + 期限预警 + 健康分异常
> - **团队负荷看板**:律师工时热力 + 任务逾期排行
