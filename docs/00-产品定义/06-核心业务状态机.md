# 第 6 章 · 核心业务状态机

> 本章定义 LM Unity · Counsel + 律时 的 **4 大核心业务状态机**:推广内容 / 线索生命周期 / 服务产品订单 / 案件办理。状态机是流程引擎、权限校验(StageGate)、合规审计的直接输入。

---

## 6.1 推广内容状态机

```
DRAFT (草稿)
  ↓
AI_RISK_CHECK (AI 风险初审)
  ↓
OPERATION_REVIEW (运营自检)
  ↓
LAWYER_REVIEW (律师专业审查)
  ↓
PARTNER_REVIEW (合伙人 / 风控审查)
  ↓
APPROVED (审核通过)
  ↓
PUBLISHED (已发布)
  ↓
TAKEDOWN (已下架) / RECTIFICATION_REQUIRED (需整改) / ARCHIVED (已归档)
```

### 状态清单

| 代码 | 名称 |
|------|------|
| `DRAFT` | 草稿 |
| `AI_RISK_CHECK` | AI 风险初审 |
| `OPERATION_REVIEW` | 运营自检 |
| `LAWYER_REVIEW` | 律师专业审查 |
| `PARTNER_REVIEW` | 合伙人 / 风控审查 |
| `APPROVED` | 审核通过 |
| `PUBLISHED` | 已发布 |
| `TAKEDOWN` | 已下架 |
| `RECTIFICATION_REQUIRED` | 需整改 |
| `ARCHIVED` | 已归档 |

### 流转规则(伪代码)

```python
def can_publish(content):
    if content.contains_result_promise:
        return block("PUBLISHED", reason="承诺办案结果")
    if content.contains_win_rate:
        return block("PUBLISHED", reason="宣示胜诉率")
    if content.contains_judicial_relationship_hint:
        return block("PUBLISHED", reason="暗示司法关系")
    if content.uses_client_case and not content.client_authorized:
        return block("PUBLISHED", reason="使用客户案例未授权")
    if content.review_status != "APPROVED":
        return block("PUBLISHED", reason="未通过审核")
    return allow("PUBLISHED")
```
