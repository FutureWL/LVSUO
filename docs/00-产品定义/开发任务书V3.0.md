## 一、产品定位

LM Unity · Counsel(智法云枢)是一款 **面向律师事务所与独立律师的一体化数字工作平台**,以数字人 **律时** 为交互入口,围绕"合规 + 增长 + 办案 + 客户成功 + 知识沉淀"五条主线,为律师群体提供端到端的执业赋能。

## 二、五大子系统(系统定位拆解)

1. **AI 智能工作平台** —— 以"律时"数字人为核心,提供智能问答、文书生成、案例检索、合同审查、法律法规查询等 AI 能力。
2. **合规增长中台** —— 律师 / 律所的市场获客、内容运营、客户旅程、私域转化、营销合规审查的运营中台。
3. **客户成功系统** —— 客户全生命周期管理、满意度回访、服务报告、续约增购、口碑裂变。
4. **案件办理操作系统** —— 案件立案、任务分配、协同办案、工时计费、文档管理、证据链、庭审辅助、合规留痕。
5. **知识经验沉淀系统** —— 内部知识库、案例库、文书模板库、判例库、团队经验复用、专家图谱。

## 三、目标用户

- **大型 / 综合性律师事务所**:跨团队协作、统一知识资产、合规风控
- **中小型 / 精品律所**:增长获客、办案效率、客户留存
- **独立律师 / 律师工作室**:个人品牌、案件全流程、轻量级 AI 助理
